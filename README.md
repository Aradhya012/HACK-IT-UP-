# Reforge AI

An autonomous, secure-by-design development platform orchestrating a multi-agent LLM workflow for vulnerability detection and automated patch generation.

## Built During
Built during **Hackzion V3** at **AMC Engineering College, Bengaluru** under the **Cybersecurity** track.

## Tech Stack
- Next.js & TypeScript
- Supabase (PostgreSQL + Auth)
- Azure OpenAI (Multi-agent orchestration)
- Azure AI Content Safety

## Features
- **Intelligent Security Scanning:** Hybrid engine using regex patterns and context-aware LLM analysis.
- **Automated Patch Generation:** AI-authored secure patches with direct GitHub PR creation.
- **Red-Team Validation Gate:** Offensive techniques to proactively validate and ensure patches are non-exploitable.
